2. Welche Daten werden erhoben?

Bei der Nutzung der App werden folgende Daten verarbeitet:

  • Anmeldedaten: E-Mail-Adresse und Name (bei Registrierung oder Apple Sign-In)
  • Turnierdaten: Turniernamen, Orte, Daten, Wetterbedingungen
  • Spielerdaten: Namen, Trikotnummern, Teamzugehörigkeit
  • Spielstatistiken: Ergebnisse, Torschützen, Spielzeiten

3. Zweck der Datenverarbeitung

Die Daten werden ausschließlich für folgende Zwecke verwendet:

  • Bereitstellung der App-Funktionen (Turnier- und Spielerverwaltung)
  • Erstellung von Spielberichten und Statistiken
  • Authentifizierung und Benutzerverwaltung

4. Datenspeicherung

Die Daten werden auf sicheren Servern in der Microsoft Azure Cloud (Rechenzentrum in Europa) gespeichert. Die Übertragung erfolgt verschlüsselt über HTTPS.

5. Weitergabe an Dritte

Deine Daten werden nicht an Dritte verkauft oder zu Werbezwecken weitergegeben. Eine Weitergabe erfolgt nur an:

  • Microsoft Azure: Hosting der Datenbank und API
  • OpenAI: Für die KI-Berichtserstellung (nur anonymisierte Spieldaten)

6. Apple Sign-In

Bei Nutzung von "Mit Apple anmelden" erhält die App nur die von dir freigegebenen Informationen (Name und E-Mail). Apple kann dabei eine private Relay-E-Mail-Adresse bereitstellen.

  • Berichtigung: Korrektur falscher Daten
  • Löschung: Löschung deiner Daten auf Anfrage
  • Datenübertragbarkeit: Export deiner Daten

Kontaktiere uns per E-Mail, um diese Rechte auszuüben.

Wir setzen technische und organisatorische Maßnahmen ein, um deine Daten zu schützen:
